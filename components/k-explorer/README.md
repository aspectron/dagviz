# kexplorer
Kaspa Explorer


## Kaspa Explorer under KDX

You can run Explorer under KDX using the following configuration entry:
```json

	"app:explorer": {
		"disable": false,
		"folder": "/path/to/k-explorer",
		"location": "http://localhost:4000",
		"args": [
			"node",
			"web-app",
			"--port=4000"
		]
	}

```
